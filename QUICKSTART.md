# ⚡ Quick Start

## 环境要求

- Node.js >= 18
- npm >= 9

---

## 1. 安装依赖

```bash
cd smart-home-3d-converted
npm install
```

---

## 2. 启动开发服务器

```bash
npm run dev
```

启动后访问：`http://localhost:5173`

局域网访问：`http://<本机IP>:5173`

---

## 3. 构建生产版本

```bash
npm run build
```

产物输出到 `dist/` 目录，可直接部署到任意静态服务器。

---

## 4. 预览构建产物

```bash
npm run preview
```

---

## 3D 场景操作

| 操作 | 效果 |
|------|------|
| 点击房间按钮 | 飞入对应房间视角 |
| 点击顶牌 | 右侧滑出设备控制面板 |
| 点击「全景」 | 返回全屋俯视视角 |
| 点击「退出全屏」 | 返回总览页 |

## 添加新设备

1. 进入「设备管理」页
2. 点击右上角「+ 新增」
3. 选择「扫描网络」（自动发现局域网设备）或「手动填写」
4. 确认后设备出现在列表中

---

## 常见问题

**Q: 3D 模型加载慢？**  
A: 模型文件较大（~50MB），首次加载需要时间，建议部署时启用 gzip 压缩。

**Q: 设备状态不同步？**  
A: 所有页面共享同一 `deviceList` 数据源，状态变更会实时反映到环境总览、设备管理和 3D 顶牌三处。

**Q: 如何修改设备数据？**  
A: 编辑 `src/views/HomeView.vue` 中的 `deviceList` ref（约第 1604 行）。
